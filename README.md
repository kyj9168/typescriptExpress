typescriptExpress study
##### PORT : 3000

---

`npm run dev`  
타입스크립트로 개발한 코드를 개발 모드로 실행하는 명령어 입니다..

`npm run build`  
타입스크립트로 개발한 코드를 컴파일 하는 명령어 입니다.

`npm start`  
컴파일한 자바스크립트 파일을 무중단 서비스 하기 위한 명령어 입니다.  
프로세스는 코어수 만큼 생성 됩니다.
